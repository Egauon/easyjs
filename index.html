Simplest Backend Service

Two Simple Functions

Ask()

Ask(context, query)
Sends a message to the backend. This message includes a context and query parameter. It is an async function that returns a value upon receiving information from the backend.

Respond()

Respond(context, funct())
Responds the queries from the frontend. When a query is received, the Respond function compares the query's context to its own. If the contexts match, the funct() is carried out and a value is returned.

One Simple Location

All of the backend is handled within the browser, meaning...

Your backend is secured using the same security techniques used as any modern browser.

No downloads of any kind are necessary. The fullstack development of your website can be done directly in the browser (no more messing with local files).
